# SAS-ZA4Tool
Save file editor for SAS: Zombie Assault 4 by Ninja Kiwi, written in Python.

---

## 🚀 Quick Start (Recommended for Most Users)

If you just want to use the editor without installing Python or dealing with code:

1.  Go to the [**Releases**](https://github.com/dstvx/SAS-ZA4Tool/releases) page.
2.  Download the latest `SASZA4Tool.exe`.
3.  Run the executable. 
    *   *Note: Your browser or Windows might flag it as "unrecognized" since it's an unsigned tool. You may need to click "Run anyway".*

---

## 🛠️ Running from Source (For Developers)

If you'd rather run the script directly using Python:

### Prerequisites
- Python 3.10 or higher
- [Steam](https://store.steampowered.com/app/678800/SAS_Zombie_Assault_4/) version of SAS:ZA 4 installed.

### Installation & Usage
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/dstvx/SAS-ZA4Tool.git
    cd SAS-ZA4Tool
    ```
2.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the editor**:
    ```bash
    python main.py
    ```

---

## ⚠️ Important Notes
- **Backup your save**: Always keep a copy of your `Profile.save` before making major changes.
- **Steam Cloud**: If Steam Cloud is active, it might overwrite your changes. It's recommended to edit while the game is closed.
